# Broken Ladder Recovery System's Inc.
## Vigil Core Engine & The 7-Sided Vortex

**Founder:** Toby J. Protsman  
**Mission:** To provide a non-stigmatizing, data-driven safety net for individuals in recovery, reducing administrative burden for counselors while providing life-saving biometric oversight.

---

### 🛡️ The 7-Sided Vortex
The heart of the Vigil UI is the **Heptagonal Vortex**. Each side represents a critical pillar of the recovery ecosystem:
1. **Biometrics (PPG/GSR):** Real-time monitoring of heart rate and stress.
2. **Respiratory Safety:** Automated detection of hypoventilation (overdose risk).
3. **ERIS Intelligence:** Environmental Risk Intelligence System for geofence safety.
4. **The Amulet Hardware:** Non-stigmatizing wearable integration.
5. **Counselor Oversight:** Direct-to-counselor automated emergency dispatch.
6. **Medicaid Pillar:** Automated transportation billing and FHIR compliance.
7. **The Community:** Peer-led support and accountability loops.

---

### 🚀 Technical Features
* **FastAPI Backend:** A high-performance Python engine handling real-time biometric logic.
* **ERIS Mapping:** Live Leaflet.js integration tracking "Hot Zone" entries in Columbus, Ohio.
* **Automated Dispatch:** Integrated SMTP notification system for immediate clinical intervention.
* **Role-Based UI:** Professional dark-mode dashboard designed for clinical environments.

---

### 🛠️ Quick Start
1. **Environment:** `source venv/bin/activate`
2. **Launch Engine:** `python3 main.py`
3. **Access UI:** Open `index.html` in Chrome.
